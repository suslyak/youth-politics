Инструкция по поиску

Простой поиск

Этот тип поиска предусматривает ввод терминов в одной поисковой строке. Поиск производится как по заглавию, так и по автору. После формирования поискового запроса и нажатия кнопки «Найти» будут выведены результаты поиска в форме списка ресурсов. При поиске по заглавию будут найдены документы, в названии которых присутствует слово, заданное в поисковой строке. При поиске по автору будут представлены материалы, отсортированные по атрибуту «Авторы».

Расширенный поиск

В отличие от простого, расширенный тип поиска предполагает фильтрацию по нескольким критериям. Он вызывается нажатием на соответствующую кнопку слева от поисковой строки. Поиск осуществляется по следующим атрибутам:

  • Автор;
  • Заглавие;
  • Год;
  • Страна;
  • Организация;
  • Язык;
  • Рубрика;
  • Тип документа;
  • Тематика;
  • Тематика (подуровни).

При поиске по автору, заглавию и году пользователь вводит необходимый запрос в строку. При фильтрации по другим критериям по каждому атрибуту выпадает список с возможностью выбора конкретного параметра. При сортировке по году могут быть установлены временные рамки (от… до…). Под рубрикой понимаются категории размещенных материалов: нормативно-правовые; информационно-аналитические; методические; статистические.

Как и в случае простого поиска, выдача результатов осуществляется в форме списка документов. Справа от перечня ресурсов продублированы фильтры расширенного поиска.

Подробнее об электронной библиотеке

Разделы

Электронная библиотека содержит следующие разделы, представленные на главной странице:

  • Коллекции. Материалы разделены на две коллекции: страны и международные организации. При нажатии кнопок «Все страны» или «Все международные организации» пользователь переходит на отдельные страницы коллекций.
  • Новости. Размещены тизеры новостей с возможностью перехода на каждую из них с помощью нажатия кнопки «Читать». В этом разделе предлагаются вниманию ежемесячные подборки свежих публикаций по различным темам, связанным с проблематикой молодежной политики.
  • Новые поступления. Представлены последние поступления в электронную библиотеку, отсортированные по дате загрузки.
  • Популярные ресурсы. Популярность документа определяется количеством его просмотров и скачиваний.
  • Сводная статистика. При нажатии соответствующей кнопки в шапке сайта осуществляется переход на страницу, где выводятся такие статистические показатели, как: общее количество электронных ресурсов; количество документов по каждому из атрибутов; количество просмотров и скачиваний каждого ресурса.

Двуязычный пользовательский интерфейс

Портал электронной библиотеки обеспечивает функцию переключения на английский язык. Англоязычная версия доступна при нажатии кнопки «Рус/Eng» на главной странице в правом верхнем углу.
